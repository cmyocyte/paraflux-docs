# Order Book / CLOB (Roadmap)

> **This product is on the roadmap and not yet available.** The CLOB will be introduced when bilateral instruments (options, variance swaps) launch.

At launch, Paraflux uses a **pool-to-peer model** for all power perps -- traders trade against the LP vault, not against each other. This provides instant liquidity without needing an order book or market makers.

A central limit order book will be introduced in a future release for bilateral instruments like options and variance swaps, where price discovery requires matching between traders.
